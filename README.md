# Product Catalog

Este repositorio contiene un proyecto de catálogo de productos en Node.js. Incluye la lógica de la aplicación y pruebas automatizadas para validar su comportamiento.

## Tecnologías

- Node.js
- npm
- Dependencias definidas en `package.json`
- Pruebas automatizadas configuradas en el proyecto

## Requisitos

- Node.js instalado (recomendado 16.x o superior)
- npm instalado

## Instalación

1. Abre una terminal en la carpeta del proyecto:
   ```bash
   cd d:\JuanD\code\product-catalog
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```

## Uso

Para ejecutar la aplicación, usa el script disponible en `package.json`. En la mayoría de los proyectos Node.js, el comando es:

```bash
npm start
```

Si la aplicación no dispone de `npm start`, se puede ejecutar directamente con Node:

```bash
node index.js
```

Ajusta el nombre del archivo de entrada según la estructura del proyecto.

## Pruebas

Para ejecutar las pruebas definidas en el proyecto, utiliza el siguiente comando:

```bash
npm test
```

Si el proyecto incluye un script de prueba diferente, revisa la sección de `scripts` en `package.json` y ejecuta el script correspondiente, por ejemplo:

```bash
npm run test
```

## Notas

- Revisa `package.json` para ver las librerías y los scripts disponibles.
- Si el proyecto incluye directorios como `src` o `tests`, allí estarán el código fuente y los casos de prueba.
